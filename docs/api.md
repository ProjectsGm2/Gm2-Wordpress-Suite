## Local Gemma

The plugin can execute a locally installed Gemma binary instead of using a remote API.

### Requirements
- Gemma inference binary available on the server (default `/usr/local/bin/gemma`).
- Gemma model file on disk.

### Configuration
1. Navigate to **Gm2 â†’ AI Settings** and select **Gemma (Local)**.
2. Enter the paths to the model and inference binary.
3. Save the settings.

### Performance considerations
Local inference consumes considerable CPU/GPU and memory resources and blocks the PHP request until completion. Avoid running it on low-powered or shared hosting environments.
